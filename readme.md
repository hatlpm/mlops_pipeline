# Modelo Predictivo de Riesgo Crediticio

Proyecto académico end-to-end de Machine Learning supervisado para anticipar
el comportamiento de pago de nuevos usuarios en una empresa financiera.

## Stack
- Python 3.12
- pandas, scikit-learn, xgboost, feature-engine
- FastAPI + Docker
- Streamlit
